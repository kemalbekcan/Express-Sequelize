### Installation

1. Clone the repo

```sh
git clone https://github.com/kemalbekcan/Express-Sequelize.git
```

2. Install NPM & YARN packages

```sh
npm & yarn install
```
3. Database

```sh
Create database named testdb in mysql database
```

4. Run

```sh
1) cd ./Client
2) npm & yarn start
```
